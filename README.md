# 🎬 ReelRole

ReelRole is a React Single Page Application (SPA) that allows users to discover trending movies, search for films, view detailed movie information, and manage a personalized watchlist.

This project demonstrates frontend architecture, API integration, state management, routing, testing, and responsive design.

---

## 🚀 Live Features

- Multi-view navigation using React Router
- Dynamic movie detail routes
- External API integration with TMDB
- Search functionality
- Global state management with React Context API
- Local component state management
- Responsive mobile-first design
- Unit testing with Vitest + React Testing Library

---

## 🛠 Tech Stack

- React
- Vite
- React Router
- Context API
- TMDB API
- Vitest
- React Testing Library
- JavaScript (ES Modules)
- CSS (mobile-first responsive layout)

---

## 📁 Application Routes

| Route | Description |
|-------|-------------|
| `/` | Trending movies homepage |
| `/discover` | Search for movies |
| `/movies/:movieId` | Dynamic movie details page |
| `/my-lists` | User watchlist page |
| `*` | 404 Not Found page |
